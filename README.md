# 🍅 PomoTerm

Un temporizador Pomodoro **responsivo** y **estético** diseñado específicamente para la terminal de Linux. 

## ✨ Características
- 🖥️ **Interfaz Fullscreen**: Se adapta dinámicamente al tamaño de tu ventana.
- ⚙️ **Interactivo**: Configura tus propios tiempos de trabajo y descanso.
- 🔔 **Notificaciones**: Integración nativa con notificaciones de escritorio.

## 📦 Instalación rápida
Copia y pega este comando en tu terminal:

```bash
git clone [https://github.com/tu-usuario/PomoTerm.git](https://github.com/tu-usuario/PomoTerm.git)
cd pomoterm
chmod +x install.sh
./install.sh
